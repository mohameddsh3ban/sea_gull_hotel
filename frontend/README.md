# Ala Carte Hotel Reservation App

A multilingual hotel reservation system built with React, Vite, and Tailwind CSS.

## Features
- Guest reservation form
- Admin dashboard for managing reservations
- Menu PDFs by cuisine
- Language support: English, French, German, Russian

## Tech Stack
- React
- Vite
- Tailwind CSS
- i18n
- Firebase

## Setup
```bash
npm install
npm run dev
